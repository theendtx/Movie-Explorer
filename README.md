# 🎬 Movie Explorer

Movie Explorer — React + TypeScript арқылы жасалған movie search web application.
Қолданушы фильмдерді іздей алады, оларды қарап шығады және favorite ретінде сақтай алады.

---

## 🚀 Features

* 🔎 Movie search (OMDb API арқылы)
* ❤️ Favorites system
* 💾 Favorites persistence (localStorage)
* 🌙 Dark / Light mode
* 📱 Responsive UI
* ⚡ Performance optimizations

---

## 🧰 Tech Stack

* React
* TypeScript
* Vite
* OMDb API
* CSS

---

## 🏗 Architecture

Project структурасы:

src/
components/
hooks/
services/
types/

Components UI-ды көрсетеді, ал business logic custom hooks ішінде орналасқан.

---

## ⚙️ Installation

Project-ті іске қосу үшін:

npm install
npm run dev

Содан кейін браузерде аш:

http://localhost:5173
